## 1. Minimum String Length After Removing Substrings (LeetCode #2696)
A problem that teaches stack simulation and greedy string reduction techniques by repeatedly eliminating adjacent removable substrings. You are given a string s consisting only of uppercase English letters. In one operation, you can remove any occurrence of the substring "AB" or "CD" from s. Repeat this operation as many times as possible.

This problem is commonly asked in interviews and helps build concepts like:
- Stack simulation
- String reduction

which are important for efficient string cleanup and comparison problems.

**Your task:** Return the minimum possible length of s after removing all such substrings, using an optimized stack-based approach.

### Examples

**Input:**
```
s = "ABFCACDB"
```
**Output:**
```
2
```

---

**Input:**
```
s = "ACBBD"
```
**Output:**
```
5
```

## 2. Copy From One Stack To Another
A problem that teaches stack operations and traversal techniques by transferring elements between two stacks. Given a stack of integers, copy all of its elements into a second, empty stack such that the relative order of elements is preserved, meaning the element on top of the original stack must also end up on top of the new stack. You may use only standard stack operations such as push, pop, peek, and isEmpty; no arrays, lists, or other auxiliary data structures are allowed.

This problem is commonly asked in interviews and helps build concepts like:
- Stack operations
- Recursion

which are important for efficient stack manipulation and data transfer problems.

**Your task:** Copy all elements from the given stack into a new stack using an optimized approach.

### Examples

**Input:**
```
st = [1,2,3,4]
```
**Output:**
```
[1,2,3,4]
```

---

**Input:**
```
st = [5,10,15]
```
**Output:**
```
[5,10,15]
```
