## 1. Valid Triangle Number (LeetCode #611)
A problem that teaches sorting and two pointer techniques by counting the number of valid triangles that can be formed from an array. Given an integer array nums, return the number of triplets that can form a valid triangle. A triangle is valid if the sum of any two sides is greater than the third side.

This problem is commonly asked in interviews and helps build concepts like:
- Sorting
- Two pointers
- Triplet counting

which are important for efficient array-processing problems.

**Your task:** Count the number of valid triangle triplets using an optimized approach.

### Examples

**Input:**
```
nums = [2,2,3,4]
```
**Output:**
```
3
```

---

**Input:**
```
nums = [4,2,3,4]
```
**Output:**
```
4
```

## 2. Pair with Target Sum
A problem that teaches the two pointer technique by finding a pair of numbers in a sorted array. Given a sorted array of integers and an integer target, determine whether there exists a pair of elements whose sum is equal to target. Return true if such a pair exists; otherwise, return false.

This problem is commonly asked in interviews and helps build concepts like:
- Two pointers
- Sorted arrays
- Efficient searching

which are important for efficient array-processing problems.

**Your task:** Find whether the sorted array contains a pair of elements whose sum equals the given target using an optimized approach.

### Examples

**Input:**
```
arr = [1, 2, 4, 6, 10], target = 8
```
**Output:**
```
true
```

---

**Input:**
```
arr = [2, 4, 6, 8], target = 5
```
**Output:**
```
false
```
