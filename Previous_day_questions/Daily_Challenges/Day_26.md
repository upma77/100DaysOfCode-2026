## 1. Sort Colors (LeetCode #75)
A problem that teaches array partitioning and two pointer techniques by sorting an array containing only three distinct values. Given an array nums containing only 0s, 1s, and 2s, sort the array in place without using any built in sorting function.

This problem is commonly asked in interviews and helps build concepts like:
- Two pointers
- Partitioning
- In place sorting

which are important for efficient array manipulation problems.

**Your task:** Sort the array containing 0s, 1s, and 2s using an optimized approach.

### Examples

**Input:**
```
nums = [2,0,2,1,1,0]
```
**Output:**
```
[0,0,1,1,2,2]
```

---

**Input:**
```
nums = [2,0,1]
```
**Output:**
```
[0,1,2]
```

## 2. Triplet Sum Check
A problem that teaches array traversal and two pointer techniques by checking whether three elements satisfy a sum relationship. Given an integer array arr[], determine whether there exist three elements such that the sum of two elements is equal to the third element.

This problem is commonly asked in interviews and helps build concepts like:
- Two pointers
- Sorting
- Triplet searching

which are important for efficient array processing problems.

**Your task:** Determine whether the array contains a valid triplet using an optimized approach.

### Examples

**Input:**
```
arr[] = [4,1,3,2,5]
```
**Output:**
```
true
```

---

**Input:**
```
arr[] = [3,4,5]
```
**Output:**
```
false
```
