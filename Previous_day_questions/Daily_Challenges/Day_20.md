## 1. Remove Linked List Elements (LeetCode #203)
A problem that teaches linked list traversal and in place node deletion by removing all nodes that match a given value. Given the head of a linked list and an integer val, remove all the nodes of the linked list that have Node.val == val, and return the new head.

This problem is commonly asked in interviews and helps build concepts like:
- Linked list traversal
- Node deletion

which are important for efficient linked list problems.

**Your task:** Remove all nodes with the given value from the linked list using an optimized approach.

### Examples

**Input:**
```
head = [1,2,6,3,4,5,6], val = 6
```
**Output:**
```
[1,2,3,4,5]
```

---

**Input:**
```
head = [7,7,7,7], val = 7
```
**Output:**
```
[]
```

---

**Input:**
```
head = [], val = 1
```
**Output:**
```
[]
```

## 2. Segregate Even and Odd Nodes in a Linked List
A problem that teaches linked list traversal and node rearrangement techniques by grouping even and odd valued nodes separately. Given the head of a linked list, rearrange the list such that all even valued nodes appear before all odd valued nodes while maintaining their original relative order.

**Note:** Do not create a new linked list. Rearrange the existing list.

This problem is commonly asked in interviews and helps build concepts like:
- Linked list traversal
- Pointer manipulation

which are important for efficient linked list operations.

**Your task:** Rearrange the linked list so that all even nodes appear before odd nodes using an optimized approach.

### Examples

**Input:**
```
17 -> 15 -> 8 -> 9 -> 2 -> 4 -> 6
```
**Output:**
```
8 -> 2 -> 4 -> 6 -> 17 -> 15 -> 9
```

---

**Input:**
```
1 -> 3 -> 5 -> 7
```
**Output:**
```
1 -> 3 -> 5 -> 7
```
