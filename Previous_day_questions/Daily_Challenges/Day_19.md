## 1. Remove Duplicates from Sorted List (LeetCode #83)
A problem that teaches linked list traversal and in-place node manipulation by removing duplicate values from a sorted linked list. Given the head of a sorted linked list, delete all duplicate nodes such that each element appears only once. Return the linked list, still sorted.

This problem is commonly asked in interviews and helps build concepts like:
- Linked list traversal
- Handling sorted data structures

which are important for efficient linked-list problems.

**Your task:** Remove the duplicate nodes from the sorted linked list using an optimized approach so that each element appears only once.

### Examples

**Input:**
```
head = [1,1,2]
```
**Output:**
```
[1,2]
```

---

**Input:**
```
head = [1,1,2,3,3]
```
**Output:**
```
[1,2,3]
```

## 2. Middle of the Linked List
A problem that teaches linked list traversal by locating the middle node of a linked list in a single pass. Given the head of a singly linked list, return the middle node of the linked list. If there are two middle nodes, return the second middle node.

This problem is commonly asked in interviews and helps build concepts like:
- Linked list traversal
- Single-pass algorithms

which are important for efficient linked-list operations.

**Your task:** Find the middle node of the linked list using an optimized approach.

### Examples

**Input:**
```
head = [1,2,3,4,5]
```
**Output:**
```
3
```

---

**Input:**
```
head = [1,2,3,4,5,6]
```
**Output:**
```
4
```
